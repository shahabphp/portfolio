# Motion Principles

Animation exists to improve communication.

Never animate simply because animation is possible.

---

## Goals

Guide attention.

Explain hierarchy.

Provide feedback.

Improve perceived quality.

Increase delight.

---

## Style

Elegant.

Smooth.

Natural.

Subtle.

Responsive.

---

## Timing

Fast interactions.

Avoid long delays.

Avoid excessive easing.

Keep the interface feeling responsive.

---

## Preferred Effects

Fade

Translate

Scale

Opacity

Layout transitions

Scroll reveal

Micro interactions

Hover feedback

---

## Avoid

Spinning elements

Random floating objects

Large parallax effects

Constant movement

Flashy transitions

Motion that distracts from content

---

## Accessibility

Respect reduced-motion preferences.

Never communicate information using animation alone.

Interfaces should remain fully usable without motion.
